# Installation

Install **TwoEasyEngine** via npm:

```bash
npm install two-easy-engine
```

Or using Yarn:

```bash
yarn add two-easy-engine
```

## Importing the library

Using ES Modules:
```js
import * as Two from "two-easy-engine";

const camera = new Two.Camera2D();
const scene = new Two.Scene();

// ...
```
